# Capturas Instagram

Renombrá y guardá acá:

| Archivo | Contenido |
|---------|-----------|
| `slide-08.png` | Adoración (arriba) + congregación / Santa Cena (abajo) → **Bethel** y Visión |

```bash
npm run crop:vision-instagram
npm run optimize:editorial
```

Genera `bethel-encuentro.jpg`, `bethel-adoracion.jpg`, etc.

También podés usar solo la captura de Bethel:

```bash
npm run crop:bethel-instagram "ruta\a\tu-captura.png"
```
